# Quizzy

Quizzy is an interactive trivia game inspired by the legendary "Kaun Banega Crorepati" (KBC). Developed using the Non-Visual Gaming Toolkit (NVGT), it provides a fully accessible experience through a robust auditory interface while offering a modern, dark-themed visual display for sighted players.

## Documentation

For a comprehensive user guide, including a full list of features, game modes, and controls, please see the [User Manual](docs/readme.md).

## Development and Customization

### Building
* Requirement: You must have the [NVGT (Non-Visual Gaming Toolkit)](https://nvgt.gg) installed to compile the source. Note: please copy the grafics plugins dll file from the lib folder to the root of your nvgt installation's lib folder , EG, C:\nvgt\lib
* Visual Functionality: Contributions to visual features may require the [Graphics Plugin by Aryan](https://github.com/aryanchoudharypro/power-plugins-for-nvgt).
